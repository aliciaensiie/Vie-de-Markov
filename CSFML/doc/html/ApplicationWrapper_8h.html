◆ isOutsideWindow()

int isOutsideWindow ( Application *  app,
RenderObject *  obj,
Vector2f *  dir 
)

vérifie si l'objet passé en argument est en dehors de la vue de la fenêtre

Parameters
app(!= NULL)
obj(!= NULL)
dir(!=NULL) (! nécessite d'être initalisé avec 0, 0 !) vecteur représentant la direction de la potentielle collision entre l'objet est les "murs" de la fenêtre. C'est-à-dire: (-1, 0) -> dépasse à gauche (1, 0) -> dépasse à droite (0, -1) -> dépasse en haut (0, 1) -> dépasse en bas Si l'objet n'est pas en dehors, alors dir n'est pas modifié.
Returns
1 si l'objet est en dehors, 0 sinon
